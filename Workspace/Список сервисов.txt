public_ws - Собирает публичные данные (тикеры, ордербук, публичные сделки) через WebSocket API ByBit. Сохраняет в Redis.
private_ws - Собирает приватные данные (пользовательские ордера, баланс) через WebSocket API ByBit с авторизацией. Сохраняет в Redis.
core - Управляет пользователями: регистрация, авторизация, выбор инструментов и стратегий. Работает с PostgreSQL.
strategies - Выполняет предопределённые адаптивные стратегии. Читает данные из Redis, создаёт/отменяет лимитные ордера. Логи в PostgreSQL.
analytics - Анализирует рыночные данные из Redis (волатильность, тренды). Формирует сигналы для Strategies. Сохраняет результаты в PostgreSQL.
notifications - Отправляет уведомления о сделках/сбоях (в будущем через Telegram). Данные из PostgreSQL/Redis.
monitoring - Отслеживает здоровье сервисов, задержки, ошибки WebSocket. Логирует в Prometheus, алерты для Grafana.
